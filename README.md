# Porosity Analysis Tool

Small Dash app to compute and visualize porosity from microscopy images (jpg/png/bmp).  
Features: legend cropping, fast mask processing, region selection, and in-memory mask caching.

1. Install dependencies
   - uv sync
     if you dont have uv install at https://docs.astral.sh/uv/

2. Run directly:
   python porosity_app.py
